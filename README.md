## Описание проекта
КупиПодариДай — это сервис вишлистов, где пользователи могут рассказать о желаемых подарках, а также скинуться на подарки для других. Проект реализован в рамках учебного курса от `Яндекс.Практикума`. В этом приложении любой пользователь может создать свой список желаний, скопировать понравившиеся подарки себе и совместно с другими собрать средства на их покупку.

В проекте реализован следующий функционал API (Бэкенд):
* **Авторизация и регистрация:** Полноценный флоу регистрации и авторизации с использованием JWT-токенов;
* **Управление пользователями:** Поиск пользователей по имени или почте, просмотр профилей других пользователей и их желаемых подарков, а также редактирование собственного профиля;
* **Управление подарками (wishes):** Добавление, редактирование и удаление собственных желаемых подарков;
* **Списки желаний (wishlists):** Создание и управление тематическими подборками подарков;
* **Сбор средств на подарки (offers):** Возможность скинуться на подарок для другого пользователя, указав сумму, с возможностью скрыть информацию о своем участии. Реализована валидация суммы сбора;
* **Копирование подарка:** Добавление чужого подарка в свой профиль одним кликом;
* **Лента подарков:** Открытый доступ к ленте из 40 последних добавленных подарков и 20 самых популярных подарков.

## Технологии
* Node.js
* NestJS
* TypeScript
* TypeORM
* PostgreSQL
* Passport.js (JWT, Local strategy)
* bcrypt (хеширование паролей)
* class-validator & class-transformer

## Развёртывание и запуск

Для работы с бэкендом локально, выполните следующие шаги:

1. Клонируйте репозиторий бэкенда:
   ```bash
   git clone https://github.com/ваш-логин/kupipodariday-backend.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd kupipodariday-backend
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Настройте подключение к базе данных PostgreSQL:
   * Создайте базу данных `kupipodariday`.
   * Создайте файл `.env` в корне проекта (в качестве примера можно использовать `.env.example`) и укажите в нём параметры подключения.
5. Запустите проект в режиме разработки:
   ```bash
   npm run start:dev
   ```
   *Бэкенд-сервис будет доступен по адресу [http://localhost:3000](http://localhost:3000).*

### Запуск фронтенда (опционально)
Для полноценной проверки функциональности API вы можете использовать готовый фронтенд:

1. Перейдите в директорию фронтенда (если он уже склонирован):
   ```bash
   cd ../kupipodariday-frontend
   ```
2. Установите зависимости:
   ```bash
   npm i
   ```
3. Запустите сборку в dev-режиме:
   ```bash
   npm start
   ```
